# Electronic Lab Notebook

Веб-приложение для управления лабораторными данными, написанное на Python с использованием фреймворка Flask.

## Описание

Приложение предоставляет функционал для:
- Управления образцами (samples)
- Управления хранилищами (storage)
- Ведения измерений (measurements)
- Ведения лабораторных экспериментов (experiments)

## Структура проекта

```
├── app.py              # Основной файл приложения
├── routes/             # Модули с обработчиками маршрутов
│   ├── samples.py      # Работа с образцами
│   ├── storage.py      # Работа с хранилищами
│   ├── measurements.py # Работа с измерениями
│   ├── experiments.py  # Работа с экспериментами
│   └── helpers.py      # Вспомогательные функции
├── templates/          # HTML-шаблоны
├── uploads/            # Папка для загруженных файлов
├── data/               # Папка для данных (JSON)
└── install.bat         # Скрипт запуска для Windows
```

## Требования

- Python 3.x
- Flask

## Установка и запуск

### На Windows

Запустите файл `ELN.bat`:

```bat
ELN.bat
```

Скрипт автоматически:
1. Проверит наличие Python
2. Создаст виртуальное окружение
3. Установит необходимые зависимости
4. Запустит сервер

### Ручная установка

1. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```

2. Активируйте виртуальное окружение:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`

3. Установите зависимости:
   ```bash
   pip install flask
   ```

4. Создайте необходимые директории:
   ```bash
   mkdir uploads data
   ```

5. Запустите приложение:
   ```bash
   python app.py
   ```

## Использование

После запуска откройте браузер и перейдите по адресу:

```
http://127.0.0.1:5000
```

## API Endpoints

- `/samples` — управление образцами
- `/storage` — управление хранилищами
- `/measurements` — управление измерениями
- `/experiments` — управление экспериментами
